## Contribution
### Code Conventions
 * Enter Bracket
 * No public variables, use properties.
 
 Auto generated:
 ```C#
 public int Variable { get; set; }
 ```
 
 User generated:
 ```C#
 private int _variable_;
 public int _Variable_
 {
     get
     {
         return value;
     }
 
      set
     {
         variable = value;
     }
 }
 ```
