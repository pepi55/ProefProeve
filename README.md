## Contribution
### Code Conventions
 * Enter Bracket
 * No public variables, use properties.
 Auto generated:
    public _Variable_ { get; set; }
 User generated:
    private _variable_;
    public _Variable_
    {
        get
        {
            return value;
        }

        set
        {
            variable = value;
        }
    }
